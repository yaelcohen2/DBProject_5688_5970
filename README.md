# DE_5786_5688_5970

## תיאור המערכת
המערכת הינה מערכת לניהול מחלקת משק בית (Housekeeping) בבית מלון, המיועדת לייעל את תהליכי הניקיון, התחזוקה והפיקוח.
המערכת מאפשרת מעקב וניהול אחר הישויות הבאות:

1.  **חדרים (`ROOM`)**: ניהול רשימת החדרים הפיזיים במלון.
2.  **משימות וסוגי ניקיון (`TASKTYPE`, `HOUSEKEPINGTASK`)**: הגדרת סוגי משימות (כגון ניקיון יסודי, סידור ערב) ושיבוץ משימות ספציפיות לחדרים, כולל תעדוף (Priority) ותאריכי יעד.
3.  **סטטוסים (`HOUSEKEEPINGSTATUS`)**: מעקב אחר סטטוס החדר/המשימה (למשל: מלוכלך, בביצוע, נקי, נבדק).
4.  **עובדים (`HOUSEKEEPINGEMPLOYEE`)**: ניהול פרטי עובדי משק הבית ומשמרותיהם.
5.  **יומן ביצוע (`CLEANNINGLOG`)**: תיעוד בזמן אמת של ביצוע המשימות, כולל שעות התחלה וסיום והערות עובדים.
6.  **בקרת איכות (`ROOMCHECK`)**: ביצוע בדיקות על ידי מפקחים, מתן ציוני איכות לניקיון ושמירת היסטוריית הבדיקות.
7.  **מלאי וחומרים (`CLEANNINGSUPPLIES`, `USES`)**: ניהול מלאי חומרי הניקוי והכלים, ומעקב אחר הכמויות שנצרכו בכל משימה.
8.  **קשרי עובדים-משימות (`BELONGSTO`)**: שיוך עובדים למשימות ספציפיות לביצוע.

המערכת מספקת תמונת מצב עדכנית על סטטוס החדרים, יעילות העובדים וניצול המשאבים במלון.
